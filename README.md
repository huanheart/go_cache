## 主要逻辑:
```go
                            是
接收 key --> 检查是否被缓存 -----> 返回缓存值 ⑴
        |  否                         是
		|-----> 是否应当从远程节点获取 -----> 与远程节点交互 --> 返回缓存值 ⑵
                |  否

```
